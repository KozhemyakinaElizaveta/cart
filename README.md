# Корзина товаров

Профильное задание
Фронтенд-разработчик в команду VK Маркет

Приложение развернуто на [GitHub Pages](https://kozhemyakinaelizaveta.github.io/vk-market).

```
git clone https://github.com/KozhemyakinaElizaveta/vk-market.git
cd vk-market
npm ci
npm start
```
##### Стек технологий
- HTML
- React
- TypeScript
- JavaScript
- Redux
